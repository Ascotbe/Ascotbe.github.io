Solr漏洞列表

windows安装

下载
Code
安装
  • 首先进入用cmd
  • 进入solr/bin
  • 分别执行start solr 以及solr start -p 8984命令即可启动成功
  • 然后访问http://localhost:8984/solr/
注意

  • 刚安装是没有Core值的需要手动添加

  • 添加的时候会报如下错误需要手动把文件拷贝过来

    • 首先在server\solr\路径创建你需要的文件夹

    • 然后把server\solr\configsets_default\conf整个文件夹拷贝到你创建的文件夹中

      bash

    • 最后这边填上你需要创建的Core，也就是你新建的那个文件名

    • 最好成功
