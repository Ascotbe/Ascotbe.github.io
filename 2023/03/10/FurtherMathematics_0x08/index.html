定积分

定义

设函数f(x)f(x)f(x)在[a,b][a,b][a,b]上有界，在[a,b][a,b][a,b]中任意插入若干个分点：

\begin{align}a=x_0 < x_1 < \cdots < x_{i-1} < x_i < \cdots < x_{n-1} < x_n=b\end{align}

把[a,b][a,b][a,b]分成nnn个小区间：

\begin{align}[x_0,x_1],\ \cdots,[x_{i-1},x_{i}],\ \cdots,\ [x_{n-1},x_n]\end{align}

各个小区间的长度依次为：

\begin{align}\Delta x_1=x_1-x_0,\ \cdots,\ \Delta x_i=x_{i}-x_{i-1},\cdots,\ \Delta x_n=x_n-x_{n-1}\end{align}

在每个小区间[xi−1,xi][x_{i-1},x_i][xi−1​,xi​]上任取一点ξi(xi−1≤ξi≤xi)\xi_i(x_{i-1}\le\xi_i\le x_i)ξi​(xi−1​≤ξi​≤xi​)，作函数值f(ξi)f(\xi_i)f(ξi​)与小区间长度Δxi\Delta x_iΔxi​的乘积f(ξi)Δxi(i=1,2,⋯ ,n)f(\xi_i)\Delta x_i(i=1,2,\cdots,n)f(ξi​)Δxi​(i=1,2,⋯,n)，并作出和：

\begin{align}S=\sum_{i=1}^{n}f(\xi_i)\Delta x_i\end{align}

记λ=max⁡{Δx1,Δx2,...,Δxn}\lambda=\max\{\Delta x_1,\Delta x_2,...,\Delta x_n\}λ=max{Δx1​,Δx2​,...,Δxn​}，如果当λ→0\lambda\to 0λ→0时，这和的极限总存在，且与闭区间[a,b][a,b][a,b]的分法及点ξi\xi_iξi​的取法无关，那么称这个极限III为函数f(x)f(x)f(x)在区间[a,b][a,b][a,b]上的定积分（Definite integral）（简称积分），记作∫abf(x)dx\int_{a}^{b}f(x)dx∫ab​f(x)dx，即：

\begin{align}\int_{a}^{b} f(x) \mathrm{~d} x=I=\lim _{\lambda\to 0} \sum_{i=1}^{n} f\left(\xi_{i}\right) \Delta x_{i}\end{align}

黎曼和

如下图所示，y=f(x)y=f(x)y=f(x)为在[a,b][a,b][a,b]上的有界函数。

在[a,b][a,b][a,b]中任意插入若干个分点：

\begin{align}a=x_0 < x_1 < \cdots < x_{i-1} < x_i < \cdots < x_{n-1} < x_n=b\end{align}

这些分点把[a,b][a,b][a,b]分成nnn个小区间，每个小区间的长度为Δxi\Delta x_iΔxi​，如下图所示（为了展示方便，下图中的分点都是均匀插入的)。

黎曼和的极限就是定积分

如果恰当地（而不是任意地）在[a,b][a,b][a,b]中插入更多的分点，那么就可以看到小矩形在不断增多，不断逼近以y=f(x)y=f(x)y=f(x)为曲边的曲边梯形，如下图所示。上述操作用代数来表示就是，记λ=max⁡{Δx1,Δx2,...,Δxn}\lambda=\max\{\Delta x_1,\Delta x_2,...,\Delta x_n\}λ=max{Δx1​,Δx2​,...,Δxn​}，不断缩小λ\lambdaλ。

随着λ\lambdaλ的缩小，如果最终这些小矩形的和存在极限，该极限就是定积分，即：

\begin{align}\int_{a}^{b} f(x) \mathrm{~d} x=I=\lim _{\lambda\to 0} \sum_{i=1}^{n} f\left(\xi_{i}\right) \Delta x_{i}\end{align}
